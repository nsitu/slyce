Web Codecs Test

This demo animates artistic cross sections of a video using WebCodecs, WebDemuxer, and Canvas.

Multiple cross sections are made from the video. Each cross section samples one row of pixels from each frame at a given location. Cross sections are animated into a new video which amounts to a strategic swap of the time and space dimensions. The strategic bit allows for a loopable back-and-forth sway.

Drag/Drop, or Browse for a Video File.
